# ウィジェットシステム

デバイス構成のレイアウトタブ（定義者向け）とプロファイルの実行画面（利用者向け）で共用するウィジェット仕様。

## メタファー

| サーフェス | メタファー |
|---|---|
| ウィジェット全体 | **オシロスコープの計測パネル** — 各信号がリアルタイムで光り、状態が一目で分かる観測面 |
| キーボードウィジェット | **押した鍵盤が光る初心者向け電子キーボード** — 押下中の鍵が発光し、velocity / pressure を直感的に表示する |
| プリミティブウィジェット | **計器盤のインジケーター** — 1信号を1ブロックで表す最小単位 |

---

## グリッドシステム

すべてのウィジェットは共通のグリッド単位 `1u`（`--widget-cell`）で寸法が決まる。

| 要素 | サイズ |
|---|---|
| プリミティブウィジェット | `2u × 2u`（正方形固定） |
| キーボードウィジェット | `N_semitones × 2u`（幅はキー数に比例、高さ固定） |
| 配列グリッドウィジェット | `columns × 1u` のセルが N 個（折り返し可） |
| グループ間ギャップ | `2u` |
| アイテム間ギャップ | `1u` |

プリミティブとキーボードは高さが同じ（2u）なので、同一グループ内に混在しても下辺が揃う。

---

## プリミティブウィジェット

1信号 = 1ブロック（`2u × 2u`）。値の変化は `just-changed` エフェクト（下記）で表現する。

### switch（bool）

| 状態 | 見せ方 |
|---|---|
| `false`（OFF） | 暗いボーダーのみ、塗りなし |
| `true`（ON） | 全面アクティブカラー + glow |
| error | 全面エラーカラー + glow |
| inactive | 全体を 45% 透過 |

### slider（int / float）

縦方向の塗りで値を表現する垂直バー。

| 種別 | 塗りの基点 | 値域 |
|---|---|---|
| unipolar | 下辺から上へ | `[0, 1]` |
| bipolar | 中央から上下へ | `[-1, +1]`（中央に軸ライン） |

### pulse（pulse）

信号発生時に全面フラッシュして減衰する（800ms）。アイドル時は暗い塗りなし状態。

### dot（float）

点の XY 位置で値を表現。1D（Y 軸のみ）と XY の 2 モード。点は 180ms トランジションで移動する。

## キーボードウィジェット

keyboard コンポーネント専用の複合ウィジェット。

```
┌─ pressed row ─────────────────────────────────────────────────┐
│  ピアノ鍵盤（白鍵 flex 幅・黒鍵は上部60%に絶対配置）          │
│  押下中の鍵 = アクティブカラー                                 │
├─ indicator row (velocity / pressure / etc.) ──────────────────┤
│  半音1個 = 1u のセルが keyCount 個並ぶ                         │
│  float → 下から積み上がるミニバー                              │
│  bool  → 中央の小ドット（ON で発光）                           │
└───────────────────────────────────────────────────────────────┘
```

- 幅 = `keyCount × 1u`（半音1個を1グリッド単位に対応させることでプリミティブと横ラインが揃う）
- 高さ = `2u`（piano 行 + indicator 行を合わせて固定高）
- children（additionals）が増えると indicator 行が増える。高さは比例して伸びる

---

## 配列グリッドウィジェット

`bool[N]` / `float[N]` 型の配列を格子状に表示する。

- セル1個 = `1u × 1u`
- `columns` パラメーターで列数を指定。折り返しで複数行になる
- bool セル: ON で全面アクティブカラー
- float セル: 下から積み上がるミニバー（中央軸ライン付き）
- ユーザーは `columns` と `gap` のみ設定する（座標は持たない）

---

## 状態エフェクト

| エフェクト | 発生タイミング | 見せ方 |
|---|---|---|
| `just-changed` | 値が変化した瞬間 | ウィジェット外周に 420ms で bloom して消えるリング |
| `selected` | ユーザーが選択中（レイアウト編集時のみ） | 2.2s 周期で呼吸する focus ring（midori-green） |

`just-changed` は実行時のリアルタイム更新で毎回発火する。`selected` はレイアウト編集時にのみ表示し、実行画面では使わない。
