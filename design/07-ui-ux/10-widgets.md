# ウィジェットシステム

デバイス構成のレイアウトタブ（定義者向け）とプロファイルの実行画面（利用者向け）で共用するウィジェット仕様。

## メタファー

| サーフェス | メタファー |
|---|---|
| ウィジェット全体 | **オシロスコープの計測パネル** — 各信号がリアルタイムで光り、状態が一目で分かる観測面 |
| キーボードウィジェット | **押した鍵盤が光る初心者向け電子キーボード** — 押下中の鍵が発光し、velocity / pressure を直感的に表示する |
| プリミティブウィジェット | **計器盤のインジケーター** — 1信号を1ブロックで表す最小単位 |

---

## スナップグリッドシステム

キャンバスは正方形を縦横等間隔に並べられるグリッドで構成される。グリッド単位を `1u` とする。

すべてのウィジェットは幅・高さともに `1u` の整数倍でなければならない。正方形にならない要素（キーボード・配列グリッド等）は、内容が収まる最小のスナップ可能サイズ（各辺を `1u` の整数倍に切り上げ）に自動調整される。

| ウィジェット | 幅 | 高さ |
|---|---|---|
| プリミティブ | `2u`（固定） | `2u`（固定） |
| キーボード | `N_semitones × 1u` | 内容が収まる最小の `Nu`（自動算出） |
| 配列グリッド | `columns × 1u` | 内容が収まる最小の `Nu`（自動算出） |

グループ間ギャップ = `2u`、アイテム間ギャップ = `1u`。

## プリミティブウィジェット

1信号 = 1ブロック（`2u × 2u`）。

### switch（bool）

| 状態 | 見せ方 |
|---|---|
| `false`（OFF） | 暗いボーダーのみ、塗りなし |
| `true`（ON） | 全面アクティブカラー + glow |

### slider（int / float）

縦方向の塗りで値を表現する垂直バー。

| 種別 | 塗りの基点 | 値域 |
|---|---|---|
| unipolar | 下辺から上へ | `[0, 1]` |
| bipolar | 中央から上下へ | `[-1, +1]`（中央に軸ライン） |

### pulse（pulse）

信号発生時に全面フラッシュして減衰する。アイドル時は暗い塗りなし状態。

### dot（float）

点の XY 位置で値を表現。1D（Y 軸のみ）と XY の 2 モード。

## キーボードウィジェット

keyboard コンポーネント専用の複合ウィジェット。

```
┌─ pressed row ──────────────────────────────────────────────────┐
│  ピアノ鍵盤（白鍵 flex 幅・黒鍵は上部に絶対配置）               │
│  押下中の鍵 = アクティブカラー                                  │
├─ indicator row（velocity / pressure 等）───────────────────────┤
│  半音1個 = 1u のセルが keyCount 個並ぶ                          │
│  float → 下から積み上がるミニバー                               │
│  bool  → 中央の小ドット（ON で発光）                            │
└────────────────────────────────────────────────────────────────┘
```

- 幅 = `keyCount × 1u`（半音1個を1グリッド単位に対応させる）
- 高さ = piano 行 + indicator 行 × 行数。スナップグリッドに収まる最小の整数 `u` に自動算出
- children（additionals）が増えると indicator 行が増え、高さが再計算される

---

## 配列グリッドウィジェット

`bool[N]` / `float[N]` 型の配列を格子状に表示する。

- セル1個 = `1u × 1u`
- `columns` パラメーターで列数を指定。超過分は自動で折り返す
- bool セル: ON で全面アクティブカラー
- float セル: 下から積み上がるミニバー（中央軸ライン付き）
- 全体の幅・高さはスナップグリッドに収まる最小サイズに自動算出

---

## 状態エフェクト

| エフェクト | 発生タイミング | 見せ方 |
|---|---|---|
| `just-changed` | 値が変化した瞬間（実行中のみ） | ウィジェット外周に短時間 bloom して消えるリング |
| `selected` | ユーザーが選択中（レイアウト編集時のみ） | 周期的に呼吸する focus ring |
| `error` | 接続されている信号パスでランタイムエラーが発生した瞬間 | ウィジェット外周にエラーカラーで短時間点滅 |

`just-changed` と `error` は実行時にのみ発火する。`selected` はレイアウト編集時にのみ表示し、実行画面では使わない。

### エラー状態の伝播

変換グラフ内でランタイムエラー（零除算・型不一致等）が発生した場合、そのエラーパスに接続している入出力ブロックのウィジェットに `error` エフェクトを発火させる。エラーは `error-path` イベント（[→ `06-error-handling.md`](../06-error-handling.md)）で通知される。
