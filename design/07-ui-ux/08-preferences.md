# Preferences 設定画面

## メタファー

| サーフェス | メタファー |
|---|---|
| 画面全体 | **作業環境の設定パネル** — 作業内容ではなく作業環境を整える場所。一度設定したら触らない |
| 一般タブ | **表示・操作の調整つまみ** — テーマや言語など見た目と操作感の調整 |
| プラグインタブ | **部品棚** — インストール済みのドライバー・デバイス種別定義 を管理する棚 |
| プラグイン追加フロー | **部品の取り寄せ** — URL を指定して外部から部品を取り寄せ、棚に追加する |

---

ダッシュボードの歯車アイコンからアクセス。**一般 / AI / プラグイン** の 3 タブ構成。

## プラグインタブ

プラグインの追加・更新・削除を行う。

```
[ + プラグインを追加 ]

┌──────────────────────────────────────────────────────┐
│ Yamaha STAGEA ELS-03        v1.0.0  @yamaha-els03    │
│ devices: yamaha-els03.yaml                           │
│                              [ 更新 ]  [ 削除 ]      │
└──────────────────────────────────────────────────────┘
```

### プラグイン追加フロー

```
[ + プラグインを追加 ] 押下
  ↓
URL 入力ダイアログ
  [ https://github.com/someone/midori-yamaha-els03 ]
  [ キャンセル ]  [ 次へ ]
  ↓
プレビュー（マニフェスト取得後）
  名前: Yamaha STAGEA ELS-03
  提供するデバイス構成: devices/yamaha-els03.yaml
  [ キャンセル ]  [ インストール ]
```

ドライバーを含むプラグイン（コードあり）の場合、プレビュー画面に権限リストを追加表示する（Phase 2）：

```
プレビュー（ドライバーあり）
  名前: MIDI Plugin
  ドライバー: midi

  ⚠️ このプラグインはコードを含みます。
  要求する権限：
    ・MIDI デバイスへのアクセス
    ・ネットワーク：なし
    ・ファイルシステム：プラグイン自身のディレクトリ（読み取り）
    ・プロセス生成：なし

  [ キャンセル ]  [ インストール ]
```

Phase 1（権限宣言なし）では「このプラグインはコードを含みます。信頼できる提供者からのみインストールしてください」の警告テキストのみ表示する。
