# ダッシュボード

## メタファー

| サーフェス | メタファー |
|---|---|
| 画面全体 | **作業台の上** — 今使いたいものをすぐ手に取れる場所。道具の整理棚ではなく「次に何をするか」の起点 |
| 最近の項目カード | **積み重ねた設計書** — 最後に触ったものが一番上にある |
| 新規作成アクション | **白紙の設計用紙** — 新しい配線を始める起点 |

---

アプリ起動時のホーム画面。右上に歯車アイコン（Preferences）とプラグインアイコンを配置する。

| 要素 | 内容 |
|---|---|
| デバイス構成を新規作成 | ドライバー選択ダイアログを表示し、選択後にデバイス構成編集画面（空の状態）へ遷移 |
| 最近のデバイス構成一覧 | 最近作成・編集したデバイス構成をカード形式で表示。クリックでデバイス構成編集画面へ遷移 |
| 変換グラフを新規作成 | 変換グラフ編集画面（空の状態）へ遷移 |
| 最近の変換グラフ一覧 | 最近作成・編集した変換グラフをカード形式で表示。クリックで変換グラフ編集画面へ遷移 |
| プロファイルを新規作成 | プロファイル詳細画面（設定タブ）へ遷移 |
| 最近のプロファイル一覧 | 最近作成・使用したプロファイルをカード形式で表示。クリックでプロファイル詳細画面へ遷移 |
| 設定（歯車アイコン） | Preferences 設定画面へ遷移 |

> **設計メモ**: 現状の「3 種類のリソース一覧」という構造はファイル種別による整理であり、慣習的なファイルマネージャーパターンに近い。「次に何をしたいか」という操作意図から設計し直す余地がある。モックアップ段階で再検討する。
