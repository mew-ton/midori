# 画面構成と遷移

## 画面マップ

```
ダッシュボード
├── デバイス構成を新規作成 ──→ デバイス構成編集画面
│                                ├── バインディングタブ
│                                ├── レイアウトタブ
│                                └── プレビュータブ（デバッグ用）
├── 最近のデバイス構成 ─────→ デバイス構成編集画面
├── 変換グラフを新規作成 ───→ 変換グラフ編集画面
├── 最近の変換グラフ ───────→ 変換グラフ編集画面
├── プロファイルを新規作成 ─→ プロファイル詳細画面（設定タブ）
├── 最近のプロファイル ────→ プロファイル詳細画面
│                                ├── プレビュータブ
│                                │     └── マッピングを編集 ──→ 変換グラフ編集画面
│                                └── 設定タブ
└── 設定（歯車）───────────→ Preferences 設定画面
                                 ├── 一般タブ
                                 ├── AI タブ
                                 └── プラグインタブ
```

---

## グローバル UI

### AI チャットパネル

全画面共通で画面右端に配置するフローティングパネル。詳細は [`08-ai.md`](../08-ai.md) を参照。

- 折りたたみ可能。折りたたみ時は右端にアイコンのみ表示
- 全画面から常にアクセス可能
- AI がファイルを保存・画面遷移した場合はパネル内にインジケーターを表示する
