# プロファイル詳細画面

## メタファー

| サーフェス | メタファー |
|---|---|
| トップ画面 | **パッチベイ** — 入力・変換・出力を左から右に並べ、信号の流れを一画面で俯瞰できる配線図 |
| デバイスエントリ | **ケーブルの接続端子** — 物理的につながっているかどうかが一目でわかる端子台の一行 |
| 実行画面 | **ステージモニター** — 演奏中に入出力の信号を手元で確認するモニター卓 |
| 実行 / 停止ボタン | **電源スイッチ** — パイプラインに通電する・遮断する |
| イベントログ | **シリアルモニター** — パイプライン内を流れるデータの時系列記録 |

---

プロファイルトップ画面をハブとし、**実行画面** と **変換グラフ編集画面** への導線が分岐する。

```text
プロファイルトップ画面
  ├── [ ▶ 実行する ] ──────→ 実行画面
  └── [ グラフを編集 ] ────→ 変換グラフ編集画面（[→ 06-graph.md](06-graph.md)）
```

## プロファイルトップ画面

```text
My Profile                                  （クリックで編集）
VRChat 演奏セット用の設定                     （クリックで編集）

 入力 [ + 追加 ]          変換グラフ             出力 [ + 追加 ]
┌──────────────────┐   ┌──────────────────┐   ┌──────────────────┐
│ ● yamaha-els03   │   │                  │   │ ● vrchat-default │
│   ELS-03 Series  │──►│ els03-to-vrchat  │──►│   127.0.0.1:9000 │
│   midi           │   │     .yaml        │   │   osc            │
│   [変更] [×]    │   │                  │   │   [変更] [×]    │
│                  │   │  [ グラフを編集 ] │   │                  │
│ ○ heart-rate     │   │                  │   │                  │
│   Polar H10      │   │                  │   │                  │
│   ble            │   │                  │   │                  │
│   [変更] [×]    │   │                  │   │                  │
└──────────────────┘   └──────────────────┘   └──────────────────┘

                        [ ▶ 実行する ]
```

### 名前・説明

インライン編集。クリックでテキストフィールドに切り替わり、フォーカスを外すと自動保存される。

### デバイスエントリ

| 要素 | 内容 |
|---|---|
| 接続状態インジケーター | ● 接続済み（緑）/ ○ 未接続（グレー）/ ⚠ ファイルが見つからない（赤） |
| id | `profiles/*.yaml` に記述した `inputs[].id` / `outputs[].id` |
| 表示名 | OS が認識しているデバイス名（MIDI デバイス名・OSC のホスト:ポート等） |
| driver | 使用するドライバー種別 |
| 変更 | デバイス構成ファイルの選択ダイアログを表示 |
| × | エントリを削除。確認ダイアログなし（Undo で復元可） |
| + 追加 | デバイス構成ファイルの選択ダイアログを表示してエントリを追加 |

接続状態はリアルタイムに更新する。デバイスが接続されると ○ → ● に自動で切り替わる。

### 変換グラフ

プロファイルに対して変換グラフは常に 1 ファイル。既存ファイルを選んで紐付ける操作は持たない。

マッパーファイル名を表示する。ファイルが存在しない場合は「（未作成）」と表示する。

### アクションボタン

| ボタン | 条件 | 遷移先 |
|---|---|---|
| ▶ 実行する | 入力・出力デバイスがすべて設定済みのとき有効 | 実行画面 |
| グラフを編集 | 常に有効 | 変換グラフ編集画面（未作成時は自動生成） |

## 実行画面

ブリッジを起動して入出力の信号をリアルタイムに確認する画面。ウィジェット仕様 → [`10-widgets.md`](10-widgets.md)

### レイアウト

画面を左右に分割し、左に入力デバイス、右に出力デバイスを表示する。各デバイスは `device_id` をヘッダーとするカードで表示し、複数デバイスは縦に並べる。

```text
┌─────────────────────────────────┬────────────────────────────────┐
│ 入力                             │ 出力                           │
│                                 │                                │
│ ┌─ yamaha-els03 ──────────────┐ │ ┌─ vrchat-default ───────────┐ │
│ │ [keyboard: upper  49keys]   │ │ │ [bool grid: upper_key 49]  │ │
│ │ [keyboard: lower  61keys]   │ │ │ [slider: expression]       │ │
│ │ [keyboard: pedal  25keys]   │ │ └────────────────────────────┘ │
│ │ [slider][slider] expression │ │                                │
│ │ [switch][switch] sustain    │ │                                │
│ └─────────────────────────────┘ │                                │
│                                 │                                │
│ ┌─ heart-rate ────────────────┐ │                                │
│ │ [slider: bpm]               │ │                                │
│ └─────────────────────────────┘ │                                │
│                                 │                                │
└─────────────────────────────────┴────────────────────────────────┘

 [ ■ 停止 ]          ▶ イベントログ    （折りたたみ）
```

各カードの内部レイアウトは、そのデバイス構成の **レイアウトタブ** で定義したウィジェット配置をそのまま表示する。定義者が決めたグループ順・アイテム順でウィジェットが並ぶ。

### リアルタイム更新

実行中、値が変化したウィジェットに `just-changed` エフェクトが発火する。変換グラフ内でランタイムエラー（零除算等）が発生した場合、そのパスに接続しているウィジェットに `error` エフェクトを発火させる。エフェクト仕様 → [`10-widgets.md`](10-widgets.md)

### ブリッジの状態表現

| 状態 | ボタン | ウィジェット表示 |
|---|---|---|
| 停止中 | `[ ▶ 実行 ]` 有効 | 静的（値なし） |
| 起動中（中間） | `[ ◌ 起動中… ]` 非活性 | 静的のまま |
| 実行中 | `[ ■ 停止 ]` 有効 | 値変化のたびに `just-changed` 発火 |
| 停止中（中間） | `[ ◌ 停止中… ]` 非活性 | エフェクトが止まり静的に戻る |
| 異常終了 | `[ ▶ 実行 ⚠ ]` 有効 | 静的。イベントログが自動展開 |

### イベントログ

デフォルト折りたたみ。展開時：

```text
 ▼ イベントログ                    [ クリア ]  [ 停止 | ● 記録中 ]

 ┌─────────────────────────────────────────────────────────────┐
 │  12:34:56.123  raw-event    [yamaha-els03] noteOn ch1 note=60│
 │  12:34:56.124  device-state [yamaha-els03] upper.60.pressed=1│
 │  12:34:56.125  signal       upper.60.pressed = true          │
 └─────────────────────────────────────────────────────────────┘
```

| 要素 | 内容 |
|---|---|
| タイムスタンプ | `HH:mm:ss.SSS` 形式 |
| イベント種別 | `raw-event` / `device-state` / `signal` / `log` / `error-path` の 5 種。種別ごとに色分け |
| ログ本文 | `raw-event` / `device-state` は `[device_id]` プレフィックス付き |
| クリアボタン | 表示中のログをすべて消去 |
| 記録停止ボタン | ログの追記を一時停止。再クリックで再開 |
| 最大保持件数 | 直近 1000 件。超過分は古い順に破棄 |
| エラー時 | 折りたたみ状態でもヘッダーを赤くして通知 |

エラー・空状態 → [`09-errors.md`](09-errors.md) を参照。
